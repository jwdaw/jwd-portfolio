# Jackson Dawson Portfolio

A personal portfolio website built with **Next.js**, **React**, and **Tailwind CSS**. This site showcases my projects, skills, and experience as a full-stack developer.

## Live Site

[View the live portfolio](https://jwdaw.github.io/jwd-portfolio)

## Tech Stack

- **Framework:** Next.js (React)
- **Styling:** Tailwind CSS
- **Language:** JavaScript / TypeScript
- **Deployment:** GitHub Pages (via GitHub Actions CI/CD)

## Features

- Project showcase with descriptions and links
- Contact page
- Responsive design for mobile and desktop
- Automated deployment pipeline via GitHub Actions

## Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- npm

### Run Locally

1. Clone the repository:

```bash
git clone https://github.com/jwdaw/jwd-portfolio.git
```

2. Navigate to the project folder:

```bash
cd jwd-portfolio
```

3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm run dev
```

5. Open your browser and visit `http://localhost:3000`
